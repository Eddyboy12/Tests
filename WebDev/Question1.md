# Question One

Consider the following function:
```js
    function isEven(i) {
        return i % 2 == 0;
    }
```
Your task is to rewrite the body of this function using bitwise operators, the following opertions are restricted from usage:
```js
 +, - , %, /, *, ^
```

Good Luck!
